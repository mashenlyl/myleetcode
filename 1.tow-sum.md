输入：一个数组nums，target

假设数组中的元素都是唯一的

输出：nums中两个元素加起来等于target的下标

如[1,2,3,4] -> target:6 -> [1,3]

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int [] result = new int[2];
        for(int i = 0; i < nums.length; i++) {
            for(int j = i + 1; j < nums.length; j++) {
                if(nums[i] + nums[j] == target) {
                    result[0] = i;
                    result[1] = j;
                    return result;
                }
            }
        }
        
        return null;
    }
}
```
